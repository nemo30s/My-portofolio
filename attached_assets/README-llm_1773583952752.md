## Project Overview

This project implements and compares three different methods for identifying aspects in text and determining sentiment toward each aspect:

1. **LexiconABSA** - Rule-based approach using spaCy and VADER sentiment lexicon
2. **TransformerABSA** - Deep learning approach using HuggingFace model via API
3. **LLMABSA** - Large language model approach using Ollama with mistral:7b

All three implementations share a unified API, making them easy to compare and swap.

**Example:**
```
Input: "The pizza was delicious but the service was terrible."

Output:
  - pizza → POSITIVE (confidence: 0.85)
  - service → NEGATIVE (confidence: 0.78)
```

## Setup Instructions

### 1. Install Dependencies

```bash
# Install all required packages
pip install -r requirements.txt

# Download spaCy language model (required for all implementations)
python -m spacy download en_core_web_sm
```

### 2. Set Up Environment Variables

Create a `.env` file in the project root:

```bash
# Required for TransformerABSA (HuggingFace API)
HUGGING_API=your_huggingface_api_key_here
```

**Note:** You only need the API key if you want to use TransformerABSA. Get one free at https://huggingface.co/settings/tokens

### 3. Install Ollama (for LLMABSA)

If you want to use the LLM-based approach:

```bash
# 1. Download and install Ollama from https://ollama.ai

# 2. Pull the mistral model
ollama pull mistral:7b

# 3. Start Ollama (run in background)
ollama serve
```

**Note:** You can use LexiconABSA and TransformerABSA without Ollama.

## Usage Examples

### Basic Usage - All Three Implementations

```python
from src.lexicon_absa import LexiconABSA
from src.transformer_absa import TransformerABSA
from src.llm_absa import LLMABSA

# Example text
text = "The food was amazing but the service was slow."

# 1. Lexicon-Based (Fast, Local)
lexicon = LexiconABSA()
results = lexicon.analyze(text)
print("Lexicon Results:")
for r in results:
    print(f"  {r.aspect}: {r.sentiment} (confidence: {r.confidence:.2f})")

# 2. Transformer-Based (Accurate, Needs API)
transformer = TransformerABSA()
results = transformer.analyze(text)
print("\nTransformer Results:")
for r in results:
    print(f"  {r.aspect}: {r.sentiment} (confidence: {r.confidence:.2f})")

# 3. LLM-Based (Flexible, Needs Ollama)
llm = LLMABSA()
results = llm.analyze(text)
print("\nLLM Results:")
for r in results:
    print(f"  {r.aspect}: {r.sentiment} (confidence: {r.confidence:.2f})")
```

### Example 3: Batch Processing

```python
from src.lexicon_absa import LexiconABSA

analyzer = LexiconABSA()

# Multiple texts
texts = [
    "The pizza was delicious.",
    "The service was terrible.",
    "Nice atmosphere and good prices."
]

# Process all at once
batch_results = analyzer.analyze_batch(texts)

for i, (text, results) in enumerate(zip(texts, batch_results)):
    print(f"\nText {i+1}: {text}")
    for r in results:
        print(f"  → {r.aspect}: {r.sentiment}")
```

### Example 5: Comparing All Three Approaches


## API Documentation

All implementations follow the same unified API defined in `src/base.py`.

### Core Classes

#### `ABSAAnalyzer` (Abstract Base Class)

All analyzer implementations inherit from this class.

**Methods:**

```python
def analyze(text: str) -> List[AspectSentiment]:
    """
    Analyze text and extract aspect-sentiment pairs.

    Args:
        text (str): Input text to analyze

    Returns:
        List[AspectSentiment]: List of detected aspects with their sentiments

    Raises:
        ValueError: If text is empty or invalid
    """
```

```python
def analyze_batch(texts: List[str]) -> List[List[AspectSentiment]]:
    """
    Analyze multiple texts.

    Args:
        texts (List[str]): List of texts to analyze

    Returns:
        List[List[AspectSentiment]]: Results for each text
    """
```

#### `AspectSentiment` (Data Class)

Represents a single aspect-sentiment pair.

**Attributes:**

```python
@dataclass
class AspectSentiment:
    aspect: str                           # The identified aspect (e.g., "camera", "battery life")
    sentiment: str                        # 'positive', 'negative', or 'neutral'
    confidence: float                     # Confidence score between 0.0 and 1.0
    text_span: Optional[Tuple[int, int]]  # Character positions in text (start, end)
```

**Example:**
```python
AspectSentiment(
    aspect="camera",
    sentiment="positive",
    confidence=0.92,
    text_span=(4, 10),
)
```
